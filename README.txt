ANIME IMPOSTEUR V14

Installation :
1. Ouvre un terminal dans ce dossier.
2. Lance : npm install
3. Lance : npm start
4. Ouvre http://localhost:3000

Mise en ligne Render :
- Build command : npm install
- Start command : npm start

Nouveautés V14 :
- Interface compacte et responsive pour PC et mobile.
- Animations de lancement, carte 3D, tour de parole, vote et élimination.
- Bouton « J’ai fini de parler » réservé au joueur actif.
- Vote sous forme de cartes de joueurs.
- Révélation des sujets et des imposteurs après le vote.
- Sons légers désactivables.
- Recherche d’images renforcée avec proxy serveur, lien direct, recherche navigateur et visuel de secours.
- 12 catégories et 541 sujets.
