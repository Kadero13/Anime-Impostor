ANIME IMPOSTEUR V16.3 — TIMMY + CARTE PERSISTANTE

INSTALLATION
1. Décompresse le ZIP.
2. Remplace entièrement les fichiers de ton ancien projet par ceux de ce dossier.
3. Ouvre un terminal dans le dossier.
4. Exécute :

npm install
npm run check
git add .
git commit -m "Ajout images Timmy et carte persistante"
git push

APRÈS LE DÉPLOIEMENT RENDER
1. Attends que le déploiement soit terminé.
2. Ferme tous les anciens onglets du jeu.
3. Ouvre https://anime-impostor.onrender.com/
4. Fais Ctrl + F5 une fois.

NOUVEAUTÉS
- 22 photos Timmy intégrées directement au projet.
- Images converties en WebP très léger : environ 300 Ko au total.
- Aucune recherche externe pour les cartes Timmy.
- Après révélation, la carte se réduit sur le côté gauche et reste visible.
- La carte reste disponible pendant la discussion, le vote et le résultat.
- La carte revient après une actualisation ou une reconnexion.
