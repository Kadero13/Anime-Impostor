ANIME IMPOSTEUR V16.2 — TOURS DE PAROLE

NOUVEAU

L’hôte peut maintenant choisir de 1 à 5 tours de parole par joueur avant le vote.
Le réglage est disponible à la création de la room et dans « Réglages » pour la prochaine manche.
Le compteur indique le numéro du tour et la position du joueur.

INSTALLATION

1. Décompresse Anime-Imposteur-V16-2-Tours.zip.
2. Remplace entièrement les fichiers de ton ancien projet par ceux du dossier extrait.
3. Dans le terminal ouvert dans le projet :

npm install
npm run check
git add .
git commit -m "Ajout du nombre de tours de parole"
git push

4. Attends le redéploiement automatique de Render.
5. Ferme les anciens onglets puis ouvre :
https://anime-impostor.onrender.com/
6. Fais Ctrl + F5 une fois pour charger la V16.2.
